Sume parțiale în vectori

Considerăm un vector cu elemente numerice. În unele probleme se cere să determinăm rapid suma elementelor din anumite secvențe date. Desigur, o soluție este parcurgerea tuturor elementelor din secvență și determinarea sumei, dar această operație are complexitatea O(n)

În asemenea situații putem folosi sumele parțiale.

Pentru aceasta vom folosi un vector auxiliar numit S[] cu urmatoarea semnificatie S[i] = X[1] + X[2] + ... + X[i]

S[i] = 0 daca i = 0

S[i] = S[i - 1] + X[i];
